## ✨ What we're building

> **FoneClaw does not just answer questions. It reads, understands, taps, swipes, and gets things done on your Android phone.**

FoneClaw turns natural language into real Android actions. Ask it to read notifications, send a message, adjust settings, summarize your screen, navigate somewhere, or run a multi-step workflow across apps - and it can operate the phone interface for you.

FoneClaw supports **120+ phone actions** across **16 categories** on **Android 9+** with **no root required**, including phone status, notifications, SMS, calls, system settings, Wi-Fi, Bluetooth, screenshots, screen reading, email, calendar, alarms, notes, maps, web tasks, workflows, app interface control, and **30+ WhatsApp actions**.

Under the hood, FoneClaw acts as a real agent layer on top of Android: it can read the UI tree, tap, swipe, fill forms, open apps, and chain actions through a Koog + LLM runtime. Read-only operations can run quickly, while state-changing or sensitive actions pass through a risk-graded approval gate.

Works across Android brands including **Samsung**, **Xiaomi**, **OnePlus**, **Google Pixel**, **Motorola**, **OPPO**, **Realme**, and more.

## ⚡ Get started

1. **Download** the Android build from [foneclaw.ai/download](https://www.foneclaw.ai/download) or the [foneclaw-android releases](https://github.com/FoneClaw-AI/foneclaw-android/tree/main/releases).
2. **Install** it on any Android 9+ device. No root access is required.
3. **Enable permissions** for Accessibility Service, microphone, and notifications.
4. **Talk to FoneClaw** with commands like:

```text
Check my phone status and summarize important notifications.
Find Italian restaurants near me and send the top pick to Sarah.
```

> ✅ Core features are free to try. No root access is required.
