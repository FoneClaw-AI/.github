## What we're building

FoneClaw turns natural language into real Android actions. Ask it to read notifications, send a message, adjust settings, summarize your screen, navigate somewhere, or run a multi-step workflow across apps - and it can operate the phone interface for you.

FoneClaw supports **120+ phone actions** across **16 categories**, works on **Android 9+**, and is designed around transparent permissions plus confirmation before sensitive actions.

Works across Android brands including **Samsung**, **Xiaomi**, **OnePlus**, **Google Pixel**, **Motorola**, **OPPO**, **Realme**, and more.

## Get started

1. Download the Android build from [foneclaw.ai/download](https://www.foneclaw.ai/download.html).
2. Install it on an Android 9+ device.
3. Enable the requested permissions, including Accessibility Service for on-screen actions.
4. Try a command like:

```text
Check my phone status and summarize important notifications.
```

Core features are free to try. No root access is required.
